# Movie Library (Личная кинотека)

**Автор:** Ахиямов Булат

## Описание
GUI-приложение на Python (Tkinter) для учета просмотренных фильмов. Позволяет добавлять фильмы, фильтровать список по жанру и году, а также сохраняет данные в формате JSON.

## Функционал
1.  **Добавление:** Название, Жанр, Год, Рейтинг (0-10).
2.  **Валидация:** Проверка на корректность ввода данных (год-число, рейтинг 0-10).
3.  **Таблица:** Отображение списка с возможностью сортировки (через фильтры).
4.  **Фильтрация:** Поиск по точному году или частичному совпадению жанра.
5.  **Хранение:** Автоматическое сохранение в `movies.json`.

## Примеры использования
1.  Запустите приложение: `python main.py`
2.  Заполните поля: Название: "Inception", Жанр: "Sci-Fi", Год: "2010", Рейтинг: "9.5"
3.  Нажмите "Добавить". Фильм появится в таблице.
4.  В поле "Фильтр" -> "Жанр" введите "Sci" и нажмите "Применить". Таблица покажет только фильмы Sci-Fi.
5.  Нажмите "Сброс", чтобы увидеть весь список.

## Требования
*   Python 3.x
*   Библиотека `tkinter` (обычно идет в комплекте с Python)
